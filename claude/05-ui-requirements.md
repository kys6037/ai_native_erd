# 05 — UI Requirements

## 페이지 구성

```
/login          로그인
/register       회원가입
/              대시보드 (프로젝트 목록) [보호됨]
/project/:id   ERD 편집 [보호됨]
```

보호된 라우트: 미인증 시 `/login`으로 리다이렉트

---

## 각 페이지 요구사항

### 로그인 / 회원가입
- 중앙 정렬 카드 레이아웃
- 폼 제출 중 로딩 표시
- 에러 메시지 인라인 표시 (toast 아님)
- 두 페이지 간 전환 링크

### 대시보드
- 프로젝트 카드 그리드 (반응형)
- 카드에 표시: 프로젝트 이름, 테이블 수, 마지막 수정 시간
- 카드 액션: 열기, 이름 변경, 삭제
- 삭제 전 확인 다이얼로그
- 빈 상태: "첫 ERD를 만들어보세요" 안내
- 로딩 중 스켈레톤 UI

### ERD 편집 페이지 (3-panel 레이아웃)

```
┌─────────────────────────────────────────────────┐
│  [← 목록]  프로젝트명    [협업자] [저장] [내보내기]  │
├──────────┬──────────────────────────────────────┤
│ 사이드바  │                                      │
│          │           캔버스                      │
│ 테이블   │                                      │
│ 목록     │                                      │
│          │                                      │
│ 선택된   │                                      │
│ 테이블   │                                      │
│ 편집기   │                                      │
├──────────┴──────────────────────────────────────┤
│  [Dialect ▼]  [DDL 생성]  [임포트]  [버전]  [사전] │
└─────────────────────────────────────────────────┘
```

## 캔버스 요구사항

### 테이블 노드 표시
- 헤더: 테이블 이름 + 색상 배경
- 본문: 컬럼 목록 (이름, 타입, PK/FK 배지)
- 선택 시: 강조 테두리

### 관계 엣지 표시
- Cardinality 레이블 (1:1, 1:N, M:N)
- 선택 시 색상 변경
- 제약 이름 표시 (있을 경우)

### 인터랙션
| 동작 | 기능 |
|------|------|
| 테이블 클릭 | 선택, 사이드바에 편집기 표시 |
| 테이블 드래그 | 위치 이동 |
| 핸들 드래그 | 관계 생성 |
| 엣지 클릭 | 관계 선택 |
| 빈 공간 클릭 | 선택 해제 |
| 빈 공간 드래그 | 캔버스 팬 |
| 스크롤 | 줌 인/아웃 |

### 자동 레이아웃
- "자동 레이아웃" 버튼 또는 Ctrl+A
- FK 관계 기반 위상 정렬 배치

## 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| Ctrl+S | 저장 |
| Ctrl+Z | Undo |
| Ctrl+Y / Ctrl+Shift+Z | Redo |
| Ctrl+K | 검색 |
| Ctrl+A | 자동 레이아웃 |
| Delete / Backspace | 선택된 항목 삭제 (확인 후) |
| Escape | 선택 해제 / 모달 닫기 |

> Delete는 입력 필드에 포커스가 없을 때만 동작

---

## 모달 / 패널 요구사항

### 버전 히스토리 모달
- 버전 목록 (번호, 메시지, 날짜)
- 각 버전: "Diff 보기", "복원" 버튼
- 새 버전 생성 폼 (메시지 입력)

### Diff 뷰어 모달
- 추가된 테이블/컬럼: 초록색
- 제거된 테이블/컬럼: 빨간색
- 수정된 항목: 노란색 + before/after
- "마이그레이션 SQL 생성" 버튼
- SQL 코드 뷰어 + 복사 버튼

### 임포트 모달 (탭 구조)
1. DDL 텍스트 — 붙여넣기 영역 + Dialect 선택
2. 파일 업로드 — 드래그&드롭
3. DB 연결 — 연결 선택 → 스키마 → 테이블 체크박스

임포트 모드: "현재 ERD 교체" vs "현재 ERD에 병합"

### 내보내기 모달
- DDL SQL (dialect 선택)
- PNG
- PDF

### 데이터 사전 모달
- 테이블 선택 드롭다운
- 테이블 수준 설명 편집
- 컬럼별 설명/표준/도메인/예시 편집
- 변경 시 자동 저장 버튼 표시

### 검색 모달 (Ctrl+K)
- 즉각 검색 (입력하는 즉시)
- 테이블, 컬럼 이름 검색
- 결과 클릭 → 해당 노드 선택 + 화면 이동

---

## 사이드바 테이블 편집기 요구사항

### Columns 탭
- 컬럼 이름, 타입, 제약 편집
- 타입 변경 시 관련 필드 동적 표시/숨김
  - VARCHAR, CHAR: length 필드
  - DECIMAL: precision, scale 필드
- FK 설정: 대상 테이블/컬럼 선택
- 컬럼 추가 / 삭제

### Indexes 탭
- 인덱스 이름, 컬럼 선택, UNIQUE 체크박스
- 컬럼 다중 선택 (순서 중요)
- 인덱스 추가 / 삭제

---

## 협업 UI

- 현재 접속자 아바타 (툴바)
- 접속자 수 3명 초과 시 "+N" 표시
- 연결 끊김 시 배너 표시 + 자동 재연결

---

## 테마

- 다크 / 라이트 전환 (토글 버튼)
- 시스템 기본값 감지
- localStorage에 설정 저장
- 전환 시 모든 컴포넌트 즉시 반영

---

## 알림 / 피드백

| 상황 | 방식 |
|------|------|
| 저장 성공 | 토스트 (2초) |
| 에러 | 토스트 (4초) |
| 삭제 확인 | 다이얼로그 |
| 로딩 | 스피너 또는 스켈레톤 |
| 폼 에러 | 인라인 메시지 |

---

## 반응형

- ERD 편집 페이지: 데스크톱 전용 (≥ 1024px)
- 대시보드: 반응형 (모바일에서 카드 1열)
- 모바일에서 편집 페이지 접근 시: "데스크톱에서 이용해주세요" 안내

---

## 자동 저장

- ERD 변경 3초 후 자동 저장 (debounce)
- 저장 중 상태 표시 (툴바)
- 저장 실패 시 에러 토스트 + 수동 저장 안내
